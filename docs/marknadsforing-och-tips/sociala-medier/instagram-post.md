# Hur du delar FOTO-3D-innehåll på Instagram

Börja med att öppna sidan på din webbplats där din Marvify FOTO-3D-visare finns tillgänglig och kopiera den fullständiga URL:en från din webbläsare. Denna URL är destinationen som tittare kan besöka efter att de sett ditt Instagram-inlägg.

Om din Marvify-visare ännu inte är konfigurerad på din webbplats kan du hänvisa till den tekniska dokumentationen och exempelguiden. Dessa resurser beskriver hur du förbereder en 3D-upplevelse som kan nås från externa plattformar.

- [Teknisk dokumentation](https://docs.marvify.io/snabbstart/marvify-viewer/)
- [Exempelguide](/exempel/webpage-integration)

Förbered sedan det media du vill publicera på Instagram. Detta kan vara en statisk bild tagen från Marvify FOTO-3D-visaren, en kort videoinspelning av interaktionen eller en annan visuell representation av 3D-innehållet sparat som en bild- eller videofil på din enhet.

Eftersom Instagram-flödesinlägg inte stöder klickbara länkar i bildtexten är det vanligaste sättet att dirigera personer till din 3D-upplevelse att placera URL:en i din profil. För att göra detta, öppna Instagram, gå till din profil, välj Redigera profil och klistra in Marvify FOTO-3D-URL:en i fältet Webbplats eller en annan lämplig länk.

Efter att länken är sparad i din profil, skapa ett nytt flödesinlägg. På Instagram, tryck på skapa-knappen och välj Inlägg. Välj den förberedda bilden eller videon som visar ditt Marvify FOTO-3D-innehåll från ditt enhetsgalleri. Instagram Stories kan också användas om du vill ha ett tillfälligt inlägg och dessa stöder att lägga till en länkklisterlapp som tittare kan trycka på för att öppna URL:en.

Om du vill visa flera vyer av samma 3D-upplevelse kan du skapa en karusell genom att välja flera bilder eller videor i samma inlägg. Instagram kommer att visa dem som svepbara objekt i flödet.

På bildtextsskärmen, skriv en beskrivning för inlägget. Du kan inkludera instruktioner om hur man kommer åt 3D-upplevelsen, till exempel genom att hänvisa till länken som är sparad i din profil eller genom att tillhandahålla URL:en i vanlig text. Användare kan kopiera URL:en manuellt om de vill öppna den i en webbläsare.

Alternativt kan du lägga till URL:en igen som en kommentar under ditt eget inlägg. Detta gör inte länken klickbar i flödet men håller adressen synlig i kommentarstråden.

När du är klar med att justera bildtexten och inställningarna, publicera inlägget till ditt Instagram-flöde. Följare som ser inlägget kan besöka din profil och använda profillänken för att komma åt Marvify FOTO-3D-sidan på din webbplats.

Instagram inkluderar ytterligare funktioner och publiceringsalternativ som inte täcks i denna översikt. För mer detaljerad information kan du hänvisa till Instagrams officiella dokumentation på [help.instagram.com](https://help.instagram.com/).

*Instagram kan uppdatera funktioner, layouter eller länkhanteringsbeteende över tid. Stegen i denna guide tillhandahålls som allmän vägledning och kanske inte alltid återspeglar den senaste versionen av appen.*
